# codex-workbench

Dedicated workspace for Codex tools, notes, and helper configuration.

## Purpose

This repository is reserved for Codex-specific setup rather than product code. Use it to track:

- installed skills and workflow extensions
- reusable prompts, notes, and operating preferences
- helper scripts or small experiments that improve day-to-day work
- setup decisions that should be easy to recover later
